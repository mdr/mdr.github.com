Graphospasm

2011Scala, Eclipse plug-ins, GEF, Maven, Tycho

 

A graph editor written using Eclipse GEF. The graph model supports attributed vertices and labelled edges. It can be also be used to visualise XML & RDF documents.

Scalariform

2009-2011Scala, sbt

 

A source code formatter for Scala, both a stand-alone tool and a library integrated with a number of editors and IDEs (Eclipse, Emacs via ENSIME). Used by a number of Scala projects, including Akka and Casbah.

https://github.com/mdr/scalariform

 

Scala IDE for Eclipse
